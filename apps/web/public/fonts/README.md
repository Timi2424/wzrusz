# TAN fonts (TanType)

Licencja komercyjna — pliki od właściciela marki. Nie redystrybuuj poza projektem Wzrusz.

| Plik | Rola CSS (`styles.scss`) |
|------|--------------------------|
| `TAN-GRANDEUR-Regular.otf` | `--font-display` — sporadyczne akcenty (np. eyebrow) |
| `TAN-BUSTER-Regular.otf` | `--font-logo` — krótkie hasła (opcjonalnie tekst) |
| `TAN-Garland-Regular.otf` | `--font-serif` — leady, opisy |

Wavy wordmark w headerze = grafika `brand/logo.png`, nie font.
